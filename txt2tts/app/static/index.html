1. 选择文件

支持哪些文件？

支持标准 Markdown（.md / .markdown）和纯文本（.txt）。流程：本地清洗（去 md 符号）→ MiniMax M3 标准化（替换 URL/代码/公式、加标点、控字数）→ MiniMax Speech 2.8 合成。

2. 选择语音

默认语音来自 MiniMax /v1/voices；若未配置 API key，会显示一组常见备用语音。

3. 预览（M3 标准化后）
